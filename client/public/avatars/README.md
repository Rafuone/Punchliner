# Avatars — dépose tes images ici

L'app charge automatiquement les images de ce dossier dans les cadres du character select.
Si une image manque, un placeholder (silhouette) s'affiche à la place.

## Nommage (2 images par rappeur)

- **Portrait (grand, buste)** → `{id}.png`  (ex. `booba.png`)
- **Avatar (vignette + petite photo)** → `{id}-face.png`  (ex. `booba-face.png`)

Formats : PNG (transparent conseillé pour le portrait) ou JPG. Le portrait s'affiche en buste, l'avatar en vignette carrée.
